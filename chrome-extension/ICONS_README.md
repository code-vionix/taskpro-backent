# Extension Icons

The extension requires 3 icon sizes. You can create simple icons or use placeholder images.

For now, you can use any 16x16, 48x48, and 128x128 PNG images and name them:
- icon16.png
- icon48.png
- icon128.png

Place them in this folder.

You can create simple icons using any image editor or online tool like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

Or use this simple placeholder approach:
1. Create a solid color square image
2. Resize to 16x16, 48x48, and 128x128
3. Save as PNG

The icons should represent monitoring/tracking (e.g., an eye icon, radar icon, or shield icon).
